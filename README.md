## Ubuntu

### Symbolic links creation

1. Change the directory to `/ubuntu/`.
2. Make the script executable:
    ```bash
    chmod +x ~/dotfiles/ubuntu/create_symlinks.sh

    ```
3. Run the script as admin:
    ```bash
    sudo ~/dotfiles/ubuntu/create_symlinks.sh
    ```
